CREATING POLYGON VECTOR LAYER 

STEP 1 : LAYER >CLICK ON CREATE LAYER > NEW SHAPED FILE LAYER > FILE NAME "Vikhroli.shp" > 
         FILE ENCODING "UTF-8" > GEOMETRY TYPE "POLYGON" > NEW FILE MAI "Name" , ADD TO FIELD
         LIST > OK

STEP 2 : SELECT VIKHROLI (MEANS CLICK ON TICKMARK ) > CLICK ON THAT YELLOW PENCIL > CLICK ON 
         THAT GREEN COLOUR POLYGON > NOW DRAW AS YOU WANT > NOW SET ID NO AND NAME [FOR EG :
         ID NO : 1 AND NAME : VIKHROLI ] OK 

STEP 3 : CLICK ON VIKHROLI > PROPERTIES > CHANGE COLOUR AS YOU WANT 


CREATING LINE VECTOR LAYER 

STEP 1 : LAYER >CLICK ON CREATE LAYER > NEW SHAPED FILE LAYER > FILE NAME "roadroutes.shp" > 
         FILE ENCODING "UTF-8" > GEOMETRY TYPE "LineString" > NEW FILE MAI "Name" , ADD TO  
         FIELD LIST > OK

STEP 2 : Do same process for line 

STEP 3 : STEP 3 : CLICK ON roadroutes > PROPERTIES > CHANGE lines AS YOU WANT
