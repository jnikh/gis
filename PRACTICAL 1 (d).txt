CALCULATING LINE LENGTH AND STATISTICS 

STEP 1 : LAYER > ADD LAYER > VECTOR LAYER > CLICK ON 3 DOT > GO TO PRACT 1 > D > DATA > 
         IND_rrd > "IND_rail.shp > ADD


STEP 2 : LAYER > ADD LAYER > VECTOR LAYER > CLICK ON 3 DOT > GO TO PRACT 1 > D > DATA > 
         IND_adm > "IND_adm0.shp > ADD

STEP 3 : DOUBLE CLICK ON IND adms0 > SELETE BORDER "OUTLINE BLUE" > APPLY > OK 

STEP 4 : CLICK ON IND rail > OPEN ATTRIBUTE TABEL > CLICK ON YELLOW PENCIL > FIELD CALCULATOR 
         > OUTPUT FIELD NAME "route_len" > OUTPUT FIELD TYPE "DECIMAL NUMBER" > PRECISION "2"
         > ROW NUMBER "GEOMETRY" > in that comment box "$length/1000" > OK > SAVE EDITS

STEP 4 : VECTOR > ANALYSIS TOOL > BASIC STRUCTURE FOR FIELD > FIELD TO CALCULATE STATISTICS 
         ON "route_len" > RUN > CLOSE 

STEP 5 : AFTER THIS PROCESS HTML FILE HAS BEEN GENERATED SO > CLICK ON IND rail PROPERTIES >
         > LABELS > SELECT "SINGLE LABELS" > VALUES > "route_len" > COLOUR RED > SIZE > APPLY

STEP 6 : AFTER APPLYING IT YOU CAN SEE CORDINATES ON MAP > TO SEE HTML FILE > SEE ON RIGHT 
         SIDE
