STEP 2 : CLICK ON glds00ag60 > PROPERTIES > SYMBOLOGY > RENDER TYPE "SINGLEBAND PSEUDOCOLOR"
         > MIN "0" > MAX "240" > INTERPOLATION "LINEAR" > CHANGE THE COLOUR RAMP > CLICK ON 
         CLASSIFY > APPLY > OK(MODIFY THIS THINGS AS PER EXAM)

STEP 3 : RASTER > RASTER CALCULATOR > OUTPUT LAYER "population_difference" > click on 3 dot to 
         save its location > now in comment box select > glds00ag60 "-" glds90ag60 > 
         OUTPUT CRS "WGS84" > OK 

STEP 4 : NOW DISSELLECT BOTH MAP > SELECT JUST POLULATION DIFFERENCE > GO TO PROPERTIES > 
         > SYMBOLOGY > RENDER TYPE "SINGLEBAND PSEUDOCOLOR"
         > MIN "-20000" > MAX "6000" > INTERPOLATION "DISCRETE" > CHANGE THE COLOUR RAMP > 
          NOW REMOVE ALL CLOUR RAMP BY USING "-" SYBMOL  
          NOW ADD AND PUT VALUE "-20000" AND IN LABEL WRITE "No Data" 
          NOW ADD AND PUT VALUE "-10" AND IN LABEL WRITE "Negative" 
          NOW ADD AND PUT VALUE "10" AND IN LABEL WRITE "Neutral" 
          NOW ADD AND PUT VALUE "6000" AND IN LABEL WRITE "Postive" 
          NOW CHOOSE COLOUR RAMP >  APPLY > OK
