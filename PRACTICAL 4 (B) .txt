# TERRIAN DATA AND HILL SHADE ANALYSIS

STEP 1 : LAYER > ADD LAYER > RASTER LAYER > PRACICAL 4 > DATA > CHOOSE "gmtEd_mea300"

STEP 2 : NOW AT DOWN SIDE THERE IS CORDINATE COLUMN WRITE THERE "86.92,27.98" , SCALE "1:900000"

STEP 3 : RASTER > EXTRACTION > CLIP RASTER BY EXTENT... 

STEP 4 : CLIPPING EXTENT "USE MAP CANVAS EXTENT" > CLIPPED (ANYWHERE YOU WANT) 
         "clippedhimalaya" > RUN > CLOSE 
 
STEP 5 : NOW JUST SELECT clippedhimalaya AND deselect other > RASTER > EXTRACTION > CONTOUR >
         SET INTERVAL "100.000000" > GIVE FILE LOCATION CONTOURS > "countourhimalaya" > RUN > 
         CLOSE 

STEP 6 : GO TO PROPERTIES OF contourhimalaya > LABEL > SINGLE LABEL > 1.2 ELEV > CHANGE COLOR

STEP 7 : RASTER > ANALYSIS > HILLSHADE... > INPUT LAYER "clipped himalaya" > Save file > 
         "hillshade" > RUN >  CLOSE
