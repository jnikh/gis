# Georeferencing Aerial Imagery

STEP 1 : WEB > OPENLAYERS PLUGINS > OPEN STREET MAP > OPEN STREET MAP 

STEP 2 : PROJECT > PROPERTIES > WGS 84 / PSEUDO-MERCATOR > APPLY > OK 

STEP 3 : VIEW > PANELS > OSM PLACE SEARCH > GATEWAY OF INDIA > ZOOM 

STEP 4 : RASTER > GEOREFRENCING > FILE > OPEN RASTER > "GATEWAY_AERIAL_IMAGERY" 

STEP 5 : EDIT > ADD POINT > SELECT ALL FOUR POINT OF GATWAY AND CHOOSE IT FROM MAP CANVAS 

STEP 6 : SETTING > TRANSFORMATION SETTING > 

STEP 7 : TRANSFORMATION TYPE : "THIN PLATE SPLINE"
         RESAMPLING METHOD : " NEAREST NEIGHBOUR "
         TARGET SRS : "EPSG:3857 - WGS 84/ PSEUDO"
         > NOW TICK ON LOAD IN Q GIS > OK > FILE > START GEOREFRENCEING 

STEP 8 : CLOSE > SAVE THE POINT
